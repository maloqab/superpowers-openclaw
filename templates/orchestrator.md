# Skills Section for Orchestrator Agents

> Copy this section into your orchestrator agent's SOUL.md file.

---

## Skills — Use Them

You have workflow skills available. Before starting any task, check if a relevant skill applies. **If a skill exists for what you're doing, use it. No exceptions.**

### Planning & design
- **brainstorming** — When a new idea or project comes in. Explore the design before delegating implementation.
- **writing-plans** — When breaking down a multi-step project into an implementation plan.

### Coordination
- **dispatching-parallel-agents** — When delegating independent tasks to multiple specialists simultaneously.
- **subagent-driven-development** — When executing plans with independent subtasks across agents.
- **requesting-code-review** — When routing completed work to a reviewer.

### Quality
- **verification-before-completion** — Before accepting any deliverable as done. Check the output, run the tests, verify the claims.
